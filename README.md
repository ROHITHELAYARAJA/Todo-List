# Todo List App

A simple, clean todo list web app built with HTML, CSS, and vanilla JavaScript.

## Features

- Add tasks with a name and due date
- Press **Enter** or click **Add Task** to add a task
- Delete tasks with one click
- Dark mode toggle (top-right corner button)
- Clean, responsive UI

## Project Structure

```
project/
├── index.html
└── TodoList/
    └── todo.js
    └── style.css
```

## How to Run

1. Download or clone this project.
2. Open `index.html` in your browser.

No build tools or installation required — it's plain HTML/CSS/JS.

## Tech Used

- HTML5
- CSS3 (Flexbox, transitions, dark theme via class toggle)
- JavaScript (DOM manipulation, array methods)
